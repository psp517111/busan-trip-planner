# 釜山自由行協作規劃原型

這是一個可直接用瀏覽器打開的前後台原型，主題是 `2026/4/26(日) ~ 2026/4/29(三)` 的釜山四天三夜行程。

## 目前支援

- 後台先編輯每日時間軸
- 同一天可混排：
  - 航班區塊
  - 交通區塊
  - 景點區塊
- 已預設：
  - Day 1 主團去程航班
  - Day 2 朋友中途加入航班
  - Day 4 回台航班
- 行程與行程中間可插入獨立交通區塊
- 前台可看總行程預覽與 NAVER Map 側欄摘要
- 朋友可從前台送出修改需求
- 後台有 AI 助理檢查時間衝突、缺少交通區塊與提醒事項

## 頁面

- `index.html`
  - 總行程預覽
  - 航班摘要
  - 每日區塊時間軸
  - NAVER Map 路線側欄
  - 新增行程 / 修改需求
- `admin.html`
  - 航班與 API 設定
  - 每日區塊編輯器
  - 區塊上下移動 / 改到別天
  - 航班 / 交通 / 景點類型切換
  - AI 路線檢查
  - 朋友回饋審核

## API 方向

- Gemini、NAVER Map、航班 API 的代理端點與金鑰配置說明在 `backend/README.md`
- 環境變數範例在 `backend/.env.example`

## 啟動方式

- 若新的終端已抓到 Node.js：`npm start`
- 若目前終端還沒抓到 PATH，可直接用：
  - `& 'C:\Program Files\nodejs\node.exe' .\server.js`

## 目前限制

- 這台環境沒有 Node.js / Python / .NET SDK
- 所以我已把前端資料模型和後端代理介面準備好，但還不能在這裡直接跑出真正的 API server
